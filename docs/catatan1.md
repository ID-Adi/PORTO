# Catatan Tema UI/UX

-----

## Warna

Monokrom / low-saturation palette

-----

## Definisi Tema

Menurut saya, UI/UX web ini memakai tema:

“minimalist technical / editorial portfolio”

dengan campuran gaya Vercel-inspired, shadcn/ui aesthetic, dan Swiss / system-based layout.

-----

## Kenapa Terasa Seperti Itu?

Karena elemen visualnya kuat di beberapa hal ini:

-----

## Grid-Based Layout yang Tegas

Struktur halaman sangat bergantung pada garis, kolom, border, dan pembagian section yang rapi. Ini memberi kesan sistematis, presisi, profesional.

-----

## Monokrom / Low-Saturation Palette

Warna dominan netral, jadi fokus pindah ke hierarki, spacing, dan tekstur, bukan ke dekorasi warna.

-----

## Pattern Halus di Dalam Blok/Div

Yang kamu sebut “arsir” itu terlihat seperti:

dot grid / radial pattern halus
diagonal hatch / repeating linear pattern

Ini dipakai sangat tipis, jadi efeknya bukan ramai, tapi memberi:

depth ringan
rasa “engineered”
nuansa dokumentasi / blueprint / interface system

-----

## Border sebagai Elemen Utama

Bukan shadow besar atau glassmorphism berat, tapi justru outline, divider, ring, dan line system. Ini bikin tampilannya terasa clean, premium, dan confident.

-----

## Nama Gaya yang Paling Cocok

Kalau harus diberi label, saya akan bilang ini kombinasi:

Neo-minimalism
Tech editorial UI
Swiss-grid inspired web design
Vercel / shadcn style interface
System-driven minimalism

-----

## Soal “Grid-Grid” dan “Arsir” yang Bikin Menarik

Daya tarik utamanya ada di kontras antara:

layout yang rigid
dengan tekstur mikro yang lembut

Jadi bukan minimalis yang kosong, tapi minimalis yang punya detail permukaan.

Makanya terasa:

modern
rapi
intelek
profesional
“designer/developer coded”

-----

## Ciri Khas Visualnya

Beberapa prinsip desain yang dipakai:

8px-ish spacing discipline
thin borders
repeatable card modules
subtle texture overlay
high whitespace
strong typographic hierarchy
components terlihat seperti design system, bukan sekadar halaman biasa

-----

## Referensi Gaya yang Sangat Dekat

Di footer situs ini bahkan ada petunjuk inspirasinya:

tailwindcss.com
ui.shadcn.com
vercel.com

Jadi feeling yang kamu suka itu memang dekat dengan ekosistem visual: Tailwind + shadcn + Vercel aesthetic.

-----

## Kalau Kamu Mau Meniru Style Ini

Fokus ke 5 hal:

Gunakan grid & border sebagai fondasi
Pakai background pattern sangat tipis
dot grid
diagonal hatch
Batasi warna
hitam, putih, zinc, slate, gray
Buat card modular
radius kecil
border tipis
shadow minimal
Jaga whitespace & tipografi
jangan terlalu padat
hierarchy jelas

-----

## Kesimpulan Singkat

Web ini bertema:

minimalist professional tech aesthetic

dengan nuansa Vercel / shadcn-inspired, Swiss-grid, dan editorial system design.
